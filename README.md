# HTML Portfolio Project

A simple multi-page HTML portfolio website.

## Overview

This project showcases a personal portfolio homepage with links to two mini-project pages:
- Movie Ranking Project
- Birthday Invite Project

It also includes separate About and Contact pages.

## Project Structure

```text
4.3 HTML Porfolio Project/
|-- index.html
|-- README.md
|-- assets/
|   `-- images/
|       |-- movie-ranking.png
|       `-- birthday-invite.png
`-- public/
    |-- about.html
    |-- birthday-invite.html
    |-- contact.html
    `-- movie-ranking.html
```

## Pages

- `index.html`
  - Main landing page
  - Introduces the portfolio owner
  - Links to project pages and profile pages
  - Displays preview images from `assets/images/`

- `public/movie-ranking.html`
  - A sample movie ranking page

- `public/birthday-invite.html`
  - A sample birthday invitation page

- `public/about.html`
  - About page

- `public/contact.html`
  - Contact details page

## Technologies Used

- HTML5

## How to Run

1. Open `index.html` in your browser.
2. Click the links to navigate between project and profile pages.

## Notes

- This is a static website (no build tools or server required).
- Image previews used on the homepage are stored in `assets/images/`.
